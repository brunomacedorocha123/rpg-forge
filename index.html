Crie, Gerencie e Jogue RPG GURPS como Nunca Antes

RPGForge é uma plataforma elegante e completa para jogadores e mestres de GURPS. Crie personagens detalhados, gerencie campanhas, faça rolagens de dados e mergulhe em aventuras épicas.

Recursos do RPGForge

Criação de Personagens

Sistema completo de criação de personagens GURPS com todas as vantagens, desvantagens, perícias e atributos. Interface intuitiva para montar seu herói passo a passo.

Rolagem de Dados

Sistema de rolagem de dados integrado com todas as opções do GURPS. Rolagens rápidas, fórmulas personalizadas e histórico completo de todas as jogadas.

Gestão de Campanhas

Crie e gerencie suas campanhas com ferramentas para mestres. Controle de sessões, NPCs, tesouros, mapas e muito mais para uma experiência completa.

Modalidades de Uso

Modo Jogador (PC)

Para os Aventureiros

  • Crie e gerencie múltiplos personagens
  • Participe de várias campanhas simultaneamente
  • Sistema de rolagem de dados integrado
  • Diário do personagem e registro de conquistas
  • Interface otimizada para dispositivos móveis

Modo Mestre (GM)

Para os Criadores de Mundos

  • Crie e gerencie campanhas completas
  • Controle de NPCs, monstros e encontros
  • Ferramentas de mapeamento e história
  • Controle de experiência e tesouros
  • Modo mestrado para sessões ao vivo
