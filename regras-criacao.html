📚 MANUAL DE CRIAÇÃO DE PERSONAGEM

Documentação completa com todas as fórmulas, cálculos e regras do Akalanata System

1. ATRIBUTOS

Os atributos são a base do personagem. Cada ponto gasto aumenta o valor em 1.

ST (Força)

Valor Fixo = 5 + ST.esferas
Testes = 40% + (ST.esferas × 3)%
Exemplo: ST 8 → Fixo: 8 | Testes: 40% + (3×3%) = 49%

DX (Destreza)

Valor Fixo = 5 + DX.esferas
Testes = 40% + (DX.esferas × 2)%
Exemplo: DX 8 → Fixo: 8 | Testes: 40% + (3×2%) = 46%

IQ (Inteligência)

Valor Fixo = 5 + IQ.esferas
Testes = 40% + (IQ.esferas × 2)%

VIGOR

Valor Fixo = 5 + VIGOR.esferas
Testes = 40% + (VIGOR.esferas × 3)%

VT (Vitalidade)

Valor Fixo = 5 + VT.esferas

2. STATUS DERIVADOS

PV - Pontos de Vida

PV = VT × 8
VT 8 → PV = 8 × 8 = 64

PF - Pontos de Fadiga

PF = VIGOR + VT
VIGOR 8, VT 8 → PF = 8 + 8 = 16

PM - Pontos de Mana

PM = VIGOR + IQ + VT
VIGOR 8, IQ 8, VT 8 → PM = 8 + 8 + 8 = 24

Dano Base

Dano = Tabela[ST.esferas]
0: 1d-3
1: 1d-2
2-3: 1d-1
4-5: 1d
6-7: 1d+1
8-9: 1d+2
10-11: 2d-1
12-13: 2d
14-15: 2d+1

3. DEFESAS

Esquiva

Esquiva = (DX% + VIGOR%)/2 + 5% + bônus - penalidade
DX% = 46%, VIGOR% = 49%, Reflexos Rápidos +5%, Carga Média -5%
= (46+49)/2 + 5 + (-5) = 47.5 + 0 = 47.5% (arredondado)

Aparar (com arma)

Aparar = DX% + (Nível da Perícia × 4%) + bônus
DX% = 46%, Espada Nv3 (+12%), Reflexos Rápidos +5%, Ataques Múltiplos +5%
= 46% + 12% + 10% = 68%

Bloqueio (com escudo)

Bloqueio = DX% + (Nível Escudo × 4%) + bônus do escudo + bônus
DX% = 46%, Escudo Nv2 (+8%), Escudo Médio (+2%), Reflexos Rápidos +5%
= 46% + 8% + 2% + 5% = 61%

4. DESLOCAMENTO E CARGA

Deslocamento Base

Andar = (DX + VIGOR) × 0.1 metros
Correr = (DX + VIGOR) × 0.3 metros
DX 8, VIGOR 8 → Soma = 16
Andar = 16 × 0.1 = 1.6m → arredonda 2m
Correr = 16 × 0.3 = 4.8m → arredonda 5m

Limites de Carga (baseado na ST)

Nível Capacidade Penalidade Esquiva Redução Deslocamento
Leve até ST × 2 kg 0% 0m
Média ST × 2 a ST × 4 kg -5% -1m (Andar) / -3m (Correr)
Pesada ST × 4 a ST × 8 kg -10% -2m (Andar) / -6m (Correr)
Limite ST × 8 a ST × 12 kg -15% -3m (Andar) / -9m (Correr)
Exemplo: ST 10 → Leve até 20kg, Média até 40kg, Pesada até 80kg, Limite até 120kg

NH da Magia

NH = IQ% + 10% + Nível da Magia
IQ% = 50%, Magia Nv3 → 50% + 10% + 3 = 63%

Requisito: Vantagem Aptidão Mágica para desbloquear escolas.

Custo: 2 PP para desbloquear magia, 2 PP para cada nível adicional.

9. CÁLCULO DO NÍVEL

Nível = Total de Esferas + 1
Esferas: ST 3, DX 2, IQ 2, VIGOR 3, VT 3 = Total 13
Nível = 13 + 1 = 14
